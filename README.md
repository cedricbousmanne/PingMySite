# Ping My Site

A simple CLI Pingdom-like written in ruby

## How to use

### Git clone

### Install required Gem

```bundle```

### dotenv (.env)

Set up a new Incoming WebHook

### Run

```ruby app.rb ping http://google.com``` --expected-status-code 200  --follow-location false```

## Technologies

* Dotenv
* Thor
* Curb
* Slack-Notifier
